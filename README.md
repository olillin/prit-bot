# P.R.I.T. Bot

This is a Discord bot developed by and for P.R.I.T. of the Software Engineering
Student Division at Chalmers.

## Installing

The bot can be added to your server by using
[this invite link](https://discordapp.com/oauth2/authorize?client_id=1344643574112714783).

## Features

The bot has many useful features that specialize in helping P.R.I.T. with their
everyday work.

## Configuration

The bot has options which can be configured with the `/config` slash command.

Each option can be `get`, `set` and `unset`. For example to set the calendar
link the command you would use is:

```plain
/config calendar set https://example.com/mycal.ics
```
